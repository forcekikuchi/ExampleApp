# 家計簿アプリ
DB接続がある家計簿アプリを作成しました  
## テーブル構造
テーブル名:savings  
id saving summary date  
  
## 概要
テーブルを日付、支出、概要で表示します。
テーブルは更新・削除・追加できるようにしました。
![イメージ](./img/image1.PNG)
![追加・更新画面](./img/image4.PNG)  
テーブル内容を価格が安い順か日付が早い順に並び変えられるようにしました  
![価格順](./img/image3.PNG)  
![日付順](./img/image2.PNG)  
  
# 追加
